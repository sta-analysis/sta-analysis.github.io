We say that E is a microset of the compact set K ⊂ ℝd if there exist sequences λn ≥ 1 and un ∈ ℝd such that (λnK + un) ∩ [0,1]d converges to E in the Hausdorff metric, and moreover, E ∩ (0,1)d ≠ ∅. The main result of the talk is to prove that for a non-empty set A ⊂ [0,d] there is a compact set K ⊂ ℝd such that the set of Hausdorff dimensions attained by the microsets of K equals A if and only if A is analytic and contains its infimum and supremum. This answers a question of Fraser, Howroyd, Käenmäki, and Yu. This is joint work with Márton Elekes and Viktor Kiss.
